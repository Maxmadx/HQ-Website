Scroll Depth

Track how far users scroll on each page (25%, 50%, 75%, 100%). See which pages actually get read vs skimmed.

Time on Page / Session Duration

How long users spend on each page and in total. Track engagement, not just arrival.

Geographic Data

Country and/or city from IP geolocation. See where your visitors are coming from on a map or table.

Device & Browser Breakdown

Parse the user-agent into mobile/tablet/desktop, OS, and browser. Already captured — just not displayed yet.

Bounce Rate

Sessions that viewed only one page before leaving. Which landing pages are losing people immediately?

F

Traffic Sources / Referrer Breakdown

Where visitors come from: direct, Google, social, other sites. Already captured in referrer field — just needs a dashboard section.

G

Click Heatmap Data

Track X/Y coordinates of every click. Build a visual heatmap overlay per page.

H

What pages does a session visit in sequence? See common paths: home → training → contact, etc.

I

Returning vs New Visitors

Use localStorage to distinguish first-time vs returning visitors across sessions.

Conversion Funnel

Define a goal funnel (e.g. home → training page → contact form → submit) and track drop-off at each step.
